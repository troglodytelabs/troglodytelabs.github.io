About

Troglodyte Labs is an independent studio focused on creating small, durable web applications and tools. We combine thoughtful design, pragmatic engineering, and clear user experiences to deliver products that last.

Focus Areas

  • Web app development with Python/Django, HTML/CSS, and lightweight JavaScript frameworks.
  • Data analysis and visualization using Pandas, NumPy, and scikit‑learn.
  • Simple, maintainable design systems for clarity and usability.

Approach

  • Scope projects tightly and deliver incrementally.
  • Use familiar, proven technologies for faster iteration and easier maintenance.
  • Prioritize user clarity over unnecessary features.

Core Skills

Python, Django, SQL, Postgres, Pandas, NumPy, scikit‑learn, HTML, CSS, basic React/TypeScript, Git. Strong communication, attention to detail, and project management skills.
